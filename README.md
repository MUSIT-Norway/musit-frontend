## Requirements

* [Node.js](http://nodejs.org/)

## Installation Steps

### Develop
1. Clone repo
2. Run `npm install`
4. Run `npm run start`
5. Open `http://localhost:8000`

### Deploy
1. Clone repo
2. Run `npm install`
3. Run `npm run build`
4. Copy files under `public` to web server (nginx/apache)
